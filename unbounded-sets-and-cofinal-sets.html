    Section 1.2 Unbounded sets and cofinal sets

    Subsection 1.2.1 Cofinal and unbounded

    We will begin this section in a more abstract setting than we ended the previous one, working with an arbitrary quasi-ordered set instead of just ordinal-valued functions.

    Definition 1.2.1.

    Let \(P\) be a set (or occasionally a proper class), while \(\leq_P\) is a quasi-order on \(P\text{.}\)
    1. A subset \(Q\) of \(P\) is \(\leq_P\)-cofinal (or just cofinal if the context is clear) in \(P\) if
      \begin{equation*} (\forall p\in P)(\exists q\in Q)\left[p\leq_P q\right]. \end{equation*}
    2. We say that a subset \(Q\) of \(P\) is \(\leq_P\)-bounded (or just bounded) in \(P\) if
      \begin{equation*} (\exists p\in P)(\forall q\in Q)\left[q\leq_P p\right], \end{equation*}
      and \(Q\) is said to be \(\leq_P\)-unbounded (or just unbounded) if it is not bounded.
    3. \(P\) is \(\leq_P\)-directed if every finite subset of \(P\) is \(\leq_P\)-bounded in \(P\text{.}\)
    4. \(q\) is a \(\leq_P\)-maximum element of \(P\) if
      \begin{equation*} (\forall p\in P)\left[p\leq_P q\right] \end{equation*}
    5. The above concepts also make sense if we replace the quasi-order \(\leq_P\) with a strict partial order \(<_P\) on \(P\text{.}\)

    Example 1.2.2.

    Suppose \((P, \leq_P)\) has a \(\leq_P\)-maximum element \(q\text{.}\) Then \(P\) is \(\leq_P\)-bounded in \((P,\leq_P)\text{,}\) while the singleton \(\{q\}\) is cofinal in \((P,\leq_P)\text{.}\) In this situation, \(P\) has no unbounded subsets, while \(P\) is \(\leq_P\)-directed for trivial reasons.
    Notice that the preceding example gives us an example where \(P\) contains a \(\leq_P\)-bounded \(\leq_P\)-cofinal subset. In fact, this sort of example is essentially unique: it not hard to show that \((P, \leq_P)\) has a maximum element if and only if it has a bounded cofinal subset. We will be working primarily with structures where this will not happen. Our next examples are typical, and takes us back to the context of the preceding section.

    Example 1.2.3.

    Let \(A=\omega\text{,}\) and let \(I=\{\emptyset\}\) be the smallest ideal on \(\omega\text{.}\) We will take \(F\) to be the set of functions from \(\omega\) to \(\omega\) (often referred to as the Baire space in the sequel), and let \(X\) be the subset of \(F\) consisting of constant functions. The set \(X\) is unbounded in \((F,\leq_I)\text{:}\) given any \(f\in F\text{,}\) if we define the function \(g\in X\) to be constant with value \(f(0)+1\text{,}\) then \(\neg(g\leq_I f)\text{.}\) On the other hand, \(X\) is not cofinal in \((F, \leq_I)\) because the identity function is not bounded modulo \(I\) by any function in \(X\text{.}\) If we replace \(I\) by the larger ideal \(J=\Fin(\omega)\) of all finite subsets of \(\omega\text{,}\) then the identity function witnesses that \(X\) is bounded in \((F, \leq_J)\text{.}\)

    Subsection 1.2.2 Cardinal Characteristics

    Our next definition introduces some major players in our discussion, defining two cardinal characteristics of makes sense in the general context of Definition [cross-reference to target(s) "def:orderstuff" missing or not unique], but we will formulate them in terms of our basic context because that is the only situation where we will use them. With that in mind, throughout the rest of this section we assume that \(I\) is a (proper) ideal on a non-empty set \(A\text{.}\)
    Let \((P, \leq_P)\) be a quasi-order without a maximum element.
    1. The cardinal \(\mathfrak b(P)\) is defined to be the least cardinality of an unbounded subset of \((P, \leq_P)\text{.}\)
    2. The cardinal \(\mathfrak d(P)\) is defined to be the least cardinality of a cofinal subset of \((P,\leq_P)\text{.}\)
    3. If \((P,\leq_P)\) is a structure of the form \((F,\leq_I)\) where \(F\subseteq\On^A\) for some set \(A\) and \(I\) is an ideal on \(A\text{,}\) then we will use the notation \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\) for the cardinals \(\mathfrak{b}(P)\) and \(\mathfrak{d}(P)\text{.}\)
    There are a couple of minor technical issues that arise when discussing unbounded sets. If \(P\) has a \(\leq_P\)-maximum element, then \(P\) will have no unbounded subsets at all and the definition of \(\mathfrak{b}(P)\) is problematic. If we find ourselves in such a situation, we will just define \(\mathfrak{b}(P)\) to be \(\infty\text{,}\) a formal symbol that is considered to be larger than any cardinal. Notice that it is possible for \(\mathfrak{b}(P)\) to be finite: this happens if and only if \(P\) is NOT \(\leq_P\)-directed. We will almost always work in directed structures without a maximum element, and this situation happens if and only the corresponding cardinal \(\mathfrak{b}(P)\) is infinite.
    These sorts of problems do not arise with \(\mathfrak{d}(P)\) as \(P\) is always cofinal in itself and so \(\mathfrak{d}_I(F)\leq |F|\) will hold automatically. It is possible for \(\mathfrak{d}(P)\) to be finite as well, and \(\mathfrak{d}(P)=1\) if and only if \(P\) has a maximum element. If \(P\) is \(\leq_P\)-directed, then \(\mathfrak{d}(P)\) is finite if and only if \(\mathfrak{d}(P)=1\text{.}\)
    If we move now to structures of the form \((F, \leq_I)\) then the following related definitions make sense:
    Assume \(I\) is an ideal on the set \(A\text{,}\) and \(F\subseteq\On^A\text{.}\)
    1. The pointwise supremum of \(F\text{,}\) denoted \(\sup(F)\text{,}\) is the function defined by
      \begin{equation*} \sup(F)(a):=\sup\{f(a):f\in F\}. \end{equation*}
    2. We say \(X\subseteq F\) is supremum-bounded in \(F\) if there exists \(g\in F\) such that \(\sup(X)\le_I g\text{.}\)
    3. We define \(\sigma(F,I)\) to be the least cardinality of a subset of \(F\) which is not supremum bounded in \(F\text{.}\) If \(F\) (and therefore every subset of \(F\)) is supremum bounded in \(F\) we let \(\sigma(F,I)=\infty\text{.}\)
    Whereas the cardinals \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\) depend only on the order-theoretic structure the \(=_I\)-equivalence classes of \(F\text{,}\) we have a different situation with the definition of \(\sigma(F, I)\text{:}\) taking the supremum of a set of functions requires us to work with actual functions instead of just equivalence classes. The following example illustrates the problem.
    Let \(A\) be \(\omega\) and let \(I\) be the ideal \(\Fin(\omega)\text{.}\) We will define two sets of functions \(F\) and \(G\) as follows. First, for any countable limit ordinal \(\alpha\) we let \(g_\alpha\) be the function on \(A\) that is constant with value \(\alpha\text{,}\) and let \(G=\{g_\alpha:\alpha<\omega_1\}\text{.}\) Next, we let \(f_\alpha = g_\alpha\) whenever \(\alpha\) is an infinite countable ordinal, and for \(\alpha = n<\omega\) we define \(f_n\) by
    \begin{equation*} \label{examplesup} f_\alpha(k) = \begin{cases} \omega_1 &\text{if $k = n$, and}\\ n &\text{otherwise,} \end{cases} \end{equation*}
    and then define \(F=\{f_\alpha:\alpha<\omega_1\}\text{.}\)
    The structures \((F, \leq_I)\) and \((G,\leq_I)\) are essentially the same modulo \(I\) as the functions \(g_\alpha\) and \(f_\alpha\) agree almost everywhere for each \(\alpha\text{.}\) The cardinal characteristics defined in Definition [cross-reference to target(s) "def:bandd" missing or not unique] will be exactly the same in each structure, as these cardinals are defined based on order-theoretic relations that depend only on the \(=_I\)-equivalence classes of functions. However, the set \(\{g_n:n<\omega\}\) is supremum bounded in \(G\text{,}\) while \(\{f_n:n<\omega\}\) is NOT supremum bounded in \(F\text{,}\) even though \(f_n =_I g_n\) for each \(n<\omega\text{.}\) This phenomenon does not occur with the other pieces of Definition [cross-reference to target(s) "def:bandd" missing or not unique].
    We mentioned earlier that the cardinal characteristics of the continuum \(\mathfrak{b}\) and \(\mathfrak{d}\) are special cases of the above definition, for if we take \(A\) to be \(\omega\text{,}\) \(I\) to be \(\Fin(\omega)\text{,}\) and \(F\) to be the Baire Space of functions mapping \(\omega\) to \(\omega\text{,}\) then \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\) are just the bounding number \(\mathfrak{b}\) and the dominating number \(\mathfrak{d}\text{.}\)

    Subsection 1.2.3 Directedness

    Many things become clearer if we strengthen our assumptions to require \(F\) to be \(<_I\)-directed. In this case, a subset \(X\) of \(F\) is \(<_I\)-cofinal in \(F\) if and only if it is \(\leq_I\)-cofinal, and this just requires the observation that in a \(<_I\)-directed \(F\text{,}\) for any \(f\in F\) there is a \(g\in F\) with \(f<_I g\text{.}\) Thus, if \(F\) is \(<_I\)-directed, we can just say “\(X\) is cofinal in \(F\)” without needing to specify which of the structures \((F, \leq_I)\) and \((F, <_I)\) we mean.
    The notions of \(<_I\)-directedness and \(\leq_I\)-directedness are closely related as well, and they coincide under under the additional assumption that for any \(f\in F\) there is a \(g\in F\) such that \(f<_I g\text{.}\) Some assumption like this is necessary, as the following example illustrates.
    Let \(A=\omega\) and \(I=\Fin(\omega)\text{.}\) Partition \(A\) into countably many disjoint infinite sets \(A_n\) for \(n<\omega\text{,}\) and let \(f_n\) be the characteristic function of \(\bigcup_{i\leq n} A_i\text{,}\) that is,
    \begin{equation*} f_n(k)= \begin{cases} 1 &\text{if $k\in A_i$ for some $i\leq n$, and}\\ 0 &\text{if not.} \end{cases} \end{equation*}
    The collection \(F= \{f_n:n<\omega\}\) is \(\leq_I\)-directed; in fact, the sequence \(\langle f_n:n<\omega\rangle\) is \(\lneqq_I\)-increasing. However, \(F\) is not \(<_I\)-directed, as no two elements of \(F\) are \(<_I\)-comparable.
    What can we say about the relationship between the cardinal \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\text{?}\) If we know that every cofinal subset of \((F,\leq_I)\) is unbounded, then it is clear that \(\mathfrak{b}_I(F)\) is at most \(\mathfrak{d}_I(F)\text{.}\) The only way this might fail is in the situation where \(F\) has a maximum element, as then \(F\) will have no unbounded subsets at all (and hence \(\mathfrak{b}_I(F)\) is set to \(\infty\) by convention) while \(\mathfrak{d}_I(F)\) will be equal to \(1\text{.}\) The following proposition records the situation in a more typical setting.
    If \(F\) is \(\leq_I\)-directed without a maximum element then
    \begin{equation*} \aleph_0\leq\mathfrak{b}_I(F)\leq\mathfrak{d}_I(F)\leq |F|. \end{equation*}
    Looking at the situation where we compare two ideals, we have the following natural result as well.
    If \(I\su J\) are ideals over \(A\) and \(F\in \On^A\text{,}\) then
    1. \(\mathfrak b_I(F)\le \mathfrak b_J(F)\text{,}\) and
    2. \(\mathfrak d_I(F)\ge \mathfrak d_J(F)\text{.}\)
    Proof. If \((F,\leq_J)\) does not have an unbounded subset, then the result follows because of our convention that \(\mathfrak{b}_J(F)\) will be set to \(\infty\text{.}\) Thus, it suffices to note that any unbounded subset of \((F,\leq_J)\) is unbounded in \((F,\leq_I)\) as well. But this is equivalent to showing that bounded subsets of \((F, \leq_I)\) are bounded in \((F,\leq_J)\text{,}\) and this follows immediately from Proposition [cross-reference to target(s) "upwithI" missing or not unique]. A similar argument shows that if a set \(X\su F\) is cofinal in \((F,\le_I)\text{,}\) then it is cofinal in the structure \((F,\le_J)\) as well. This immediately implies \(\mathfrak{d}_I(F)\leq\mathfrak{d}_J(F)\text{.}\) ◻
    If \((F,\leq_I)\) is directed without a maximum element, then \(\mathfrak{b}_I(F)\) is an infinite regular cardinal.
    Proof. Let \(\lambda=\mathfrak{b}_I(F)\text{.}\) Our assumptions imply that \(\lambda\) is an infinite cardinal, so we need only worry about proving \(\lambda\) is regular. It suffices to prove that a union of fewer than \(\lambda\) subsets of \(F\) each of cardinality less than \(\lambda\) must be bounded. So suppose \(\kappa<\lambda\) and for each \(\alpha<\kappa\) let \(G_\alpha\) be a subset of \(F\) of cardinality less than \(\lambda\text{.}\) Since each \(G_\alpha\) is bounded, we can find a function \(g_\alpha\) in \(F\) witnessing this. The collection \(\{g_\alpha:\alpha<\kappa\}\) is also bounded in \(F\text{,}\) and any function \(f\in F\) that \(\leq_I\)-bounds \(\{g_\alpha:\alpha<\kappa\}\) will bound \(\bigcup_{\alpha<\kappa}G_\alpha\text{.}\) ◻
    If we strengthen the assumption of Proposition [cross-reference to target(s) "prop:firstb" missing or not unique] and require \(<_I\)-directness, then we can find unbounded subsets of \(F\) that can be enumerated as a \(<_I\)-increasing sequences of length \(\mathfrak{b}_I(F)\text{.}\) In fact, though, the following proposition actually shows us something slightly stronger.
    If \(F\) is \(<_I\)-directed then for any cofinal subset \(X\) of \(F\) there is a \(<_I\)-increasing sequence \(\langle f_\alpha:\alpha<\mathfrak{b}_I(F)\rangle\) of elements of \(X\) that is unbounded in \(F\text{.}\)
    Proof. Our hypothesis implies that \(F\) is \(\leq_I\)-directed without a maximum element, so \(\lambda=\mathfrak{b}_I(F)\) is an infinite regular cardinal by Proposition [cross-reference to target(s) "prop:firstb" missing or not unique]. Let \(\{g_\alpha:\alpha<\lambda\}\) be an unbounded subset of \(F\text{,}\) and let \(X\) be cofinal in \(F\text{.}\) We will define functions \(f_\alpha\in X\) by induction on \(\alpha<\lambda\text{,}\) making sure to arrange both \(g_\alpha<_I f_\alpha\) and that \(f_\beta<_I f_\alpha\) for \(\beta<\alpha\text{.}\) This is straightforward to do: at a stage \(\alpha\) we will have the set \(\{f_\beta:\beta<\alpha\}\cup\{g_\alpha\}\) at our disposal, and this set is bounded modulo \(I\) by some function \(h\in F\) because it is of cardinality less than \(\lambda\text{.}\) Since \(F\) is \(<_I\)-directed and \(X\) is cofinal in \(F\text{,}\) we can find \(f_\alpha\in X\) with \(h<_I f_\alpha\text{.}\) Now the sequence \(\langle f_\alpha:\alpha<\lambda\rangle\) is a \(<_I\)-increasing sequence of functions from \(X\text{,}\) and it is unbounded in \(F\) because \(Y\) was. ◻
    Let \(A\) be \(\omega\text{,}\) and let \(\{X_n:n<\omega\}\) enumerate the finite subsets of \(\omega\text{.}\) Given \(n<\omega\text{,}\) we let \(f_n\in\On^A\) be the function that is constant with value \(n\text{,}\) and define another function \(h_n\in\On^A\) by
    \begin{equation*} h_n(k) = \begin{cases} n &\text{if $k\notin X_n$, and}\\ \omega &\text{if $k\in X_n$.} \end{cases} \end{equation*}
    Thus, the functions \(f_n\) and \(h_n\) agree and have value \(n\) outside of \(X_n\text{,}\) while on the set \(X_n\) the function \(h_n\) takes value \(\omega\text{.}\) Let \(F\) and \(H\) be the corresponding sets of functions. If we let \(I = \{\emptyset\}\text{,}\) then \(F\) is \(<_I\)-increasing and hence \(<_I\)-directed. On the other hand, \(H\) is not \(<_I\)-directed: if \(X_n\neq\emptyset\) then there is no \(g\in H\) with \(f_n<_I g\text{.}\)
