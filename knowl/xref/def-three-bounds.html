Definition 2.1.1.

Suppose \(F\subseteq \On^A\) is a set of functions, and \(f\) is an upper bound for \(F\) modulo \(I\text{.}\)
  1. We say \(f\) is a minimal upper bound for \(F\) if \(g<_I f\) implies that \(g\) is not a \(\leq_I\)-upper bound for \(F\text{.}\)
  2. We say \(f\) is a least upper bound for \(F\) if \(f\leq_I g\) whenever \(g\) is a \(\leq_I\)-upper bound for \(F\text{.}\) We say that \(f\) is a lub for \(F\) modulo \(I\).
  3. (Definition 1.4.7) We say \(f\) is an exact upper bound for \(F\) if for any \(g<_I f\) there is an \(h\in F\) with \(g\leq_I h\text{.}\) We say that \(f\) is an eub for \(F\) modulo \(I\).
