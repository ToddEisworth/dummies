Theorem 2.1.4.

Suppose \(A\) is a set, \(f:A\rightarrow\On\text{,}\) \(I\) is a proper ideal on \(A\text{,}\) and \(F\subseteq\prod f\text{.}\)
  1. \(f\) is a \(<_I\)-minimal upper bound for \(F\) if and only if \(F\) is unbounded in \((\prod f, \leq_I)\text{.}\)
  2. \(f\) is a least upper bound for \(F\) if and only if \(F\) is everywhere unbounded in \((\prod f, \leq_I)\text{,}\) that is \(F\) is unbounded modulo \(I\restriction B\) for any \(I\)-positive subset \(B\) of \(A\text{.}\)
  3. \(f\) is an exact upper bound for \(F\) if and only if \(F\) is cofinal in \(\prod g\text{.}\)
  4. If in addition \(F\) is \(\leq_I\)-directed, then \(f\) is an exact upper bound for \(F\) if and only if \(F\) is unbounded in \((\prod f, \leq_J)\) for any proper ideal \(J\) on \(A\) extendingÂ \(I\text{.}\)
