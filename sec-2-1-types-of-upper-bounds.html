    Section 2.1 Types of Upper Bounds

    Subsection 2.1.1 Basic Definitions

    We start our project by looking at how exact upper bounds compare to other natural candidates for being a “smallest” upper bound modulo \(I\) for a collection of functions \(F\text{.}\) We gather a few of these possibilities in the next definition.

    Definition 2.1.1.

    Suppose \(F\subseteq \On^A\) is a set of functions, and \(f\) is an upper bound for \(F\) modulo \(I\text{.}\)
    1. We say \(f\) is a minimal upper bound for \(F\) if \(g<_I f\) implies that \(g\) is not a \(\leq_I\)-upper bound for \(F\text{.}\)
    2. We say \(f\) is a least upper bound for \(F\) if \(f\leq_I g\) whenever \(g\) is a \(\leq_I\)-upper bound for \(F\text{.}\) We say that \(f\) is a lub for \(F\) modulo \(I\).
    3. (Definition 1.4.7) We say \(f\) is an exact upper bound for \(F\) if for any \(g<_I f\) there is an \(h\in F\) with \(g\leq_I h\text{.}\) We say that \(f\) is an eub for \(F\) modulo \(I\).
    Note that these three conditions all coincide when \(I\) is a maximal proper ideal on \(A\text{,}\) and it should be clear that a least upper bound for \(F\) modulo \(I\) is also a minimal upper bound. In the circumstances of most interest to us, an exact upper bound will also be a least upper bound, as shown by the following claim.

    Proposition 2.1.2.

    If \(F\) is non-empty and \(<_I\)-directed, then an exact upper bound for \(F\) modulo \(I\) is also a least upper bound for \(F\) modulo \(I\text{.}\)

    Proof.

    Suppose that \(f\) is an exact upper bound for \(F\) modulo \(I\text{,}\) and \(g\) is any other upper bound. We need to show \(f\leq_I g\) holds.
    By replacing \(g\) with \(\min\{f, g\}\) if necessary, we may assume that \(f\leq g\text{.}\) Now suppose that \(f\nleq_I g\text{,}\) so that there is an \(I\)-positive set of \(a\in A\) with \(g(a)<f(a)\text{.}\) Let \(g'\) be the function in \(\prod f\) that agrees with \(g\) at any \(a\) for which \(g(a)\) is less than \(f(a)\text{,}\) and which takes on the value \(0\) otherwise. Since we assumed that \(F\) is \(<_I\)-directed, it follows that \(f(a)\) must be non-zero almost everywhere and therefore \(g'<_I f\text{.}\) Since \(f\) is an exact upper bound for \(F\text{,}\) this means there must be an \(h\in F\) such that \(g'\leq_I h\text{,}\) and since \(F\) is \(<_I\)-directed, we may assume that \(g'<_I h\text{.}\)
    Now we have a problem. On one hand, we know \(g'(a)<h(a)\) for almost every \(a\in A\text{,}\) and we know \(h(a)\leq g(a)\) for almost every \(a\in A\) (as \(g\) is an upper bound for \(f\)). On the other hand, \(g'(a)=g(a)\) on an \(I\)-positive set by our construction. Thus, there must be an \(a\in A\) for which \(g(a)=g'(a)<h(a)<g(a)\text{,}\) and this is a contradiction. ◻
    Some assumption like the above is necessary to conclude that an exact upper bound is a least upper bound: see Exercise [cross-reference to target(s) "exer:eubnotlub" missing or not unique] below. These varieties of upper bounds do, however, all interact with the equivalence relations \(=_I\) and \(\equiv_I\) in the way we might wish.

    Proposition 2.1.3.

    Suppose \(I\) is an ideal on the set \(A\text{,}\) and let \(F\) and \(G\) be subsets of \(\On^A\) with \(F\equiv_I G\text{.}\) If \(f\) and \(g\) are functions with \(f=_I g\text{,}\) then
    1. \(f\) is an upper bound for \(F\) modulo \(I\) if and only if \(g\) is an upper bound for \(G\) modulo \(I\text{,}\)
    2. \(f\) is a minimal bound of \(F\) modulo \(I\) if and only if \(g\) is a minimal bound of \(G\) modulo \(I\text{,}\)
    3. \(f\) is a least upper bound for \(F\) modulo \(I\) if and only if \(g\) is a least upper bound for \(G\) modulo \(I\text{,}\) and
    4. \(f\) is an exact upper bound for \(F\) modulo \(I\) if and only if \(g\) is an exact upper bound for \(G\) modulo \(I\text{.}\)
    Sometimes it is helpful to view these concepts in their contrapositive formulations. If \(f\) is a minimal upper bound for \(F\) modulo \(I\text{,}\) then any function that is almost everywhere strictly less than \(f\) will fail to bound \(F\text{.}\) If \(f\) is a least upper bound for \(F\) modulo \(I\text{,}\) then any function that is strictly less than \(f\) on an \(I\)-positive set will fail to bound \(F\text{.}\) Finally, any function that is almost everywhere strictly less than an exact upper bound \(f\) is actually bounded by some member of \(F\) modulo \(I\text{.}\) In a similar vein, we can characterize the concepts from Definition 2.1.1 for \(F\subseteq\prod f\) by the way in which \(F\) sits inside of \(\prod f\text{.}\)

    Theorem 2.1.4.

    Suppose \(A\) is a set, \(f:A\rightarrow\On\text{,}\) \(I\) is a proper ideal on \(A\text{,}\) and \(F\subseteq\prod f\text{.}\)
    1. \(f\) is a \(<_I\)-minimal upper bound for \(F\) if and only if \(F\) is unbounded in \((\prod f, \leq_I)\text{.}\)
    2. \(f\) is a least upper bound for \(F\) if and only if \(F\) is everywhere unbounded in \((\prod f, \leq_I)\text{,}\) that is \(F\) is unbounded modulo \(I\restriction B\) for any \(I\)-positive subset \(B\) of \(A\text{.}\)
    3. \(f\) is an exact upper bound for \(F\) if and only if \(F\) is cofinal in \(\prod g\text{.}\)
    4. If in addition \(F\) is \(\leq_I\)-directed, then \(f\) is an exact upper bound for \(F\) if and only if \(F\) is unbounded in \((\prod f, \leq_J)\) for any proper ideal \(J\) on \(A\) extending \(I\text{.}\)

    Proof.

    Proof. Statement (1) and (3) are very easy, and follow directly from the corresponding definitions. For (2), suppose first that there is an \(I\)-positive \(B\subseteq A\) and a function \(h\in\prod f\) such that \(h\) bounds \(F\) modulo the ideal \(I\restriction B\text{.}\) Define a function \(g\) with domain \(A\) by
    \begin{equation*} g(a)= \begin{cases} h(a) &\text{if $a\in B$, and}\\ f(a) &\text{otherwise.} \end{cases} \end{equation*}
    The function \(g\) is an upper bound for \(F\) modulo \(I\text{,}\) and since \(\neg f\leq_I g\) it follows that \(f\) is not a least upper bound for \(F\text{.}\)
    If on the other hand \(f\) is not a least upper bound for \(F\subseteq\prod f\text{,}\) then there will be a witnessing function \(g\text{,}\) and we can take
    \begin{equation*} B:=<(g, f)=\{a\in A: g(a)<f(a)\}. \end{equation*}
    The set \(B\) is \(I\)-positive (as \(g\) and \(f\) are not equal modulo \(I\)), and \(g\) bounds \(F\) modulo \(I\restriction B\text{.}\)
    As for (4), note that if \(F\) is \(\leq_I\)-cofinal in \(\prod f\) then this will remain true in \((\prod f, \leq_J)\) for any ideal \(J\) on \(A\) that extends \(I\text{.}\) If \(F\) is NOT \(\leq_I\)-cofinal in \(\prod f\) then this will be witnessed by a function \(h\in\prod f\) such that \(h\) is not bounded by any \(g\in F\text{.}\) Let \(J\) be the ideal on \(A\) generated by \(I\) together with the sets \(<(h, g)\) for \(g\in F\text{.}\) It is clear that \(F\) will be \(\leq_J\)-bounded by \(h\text{,}\) so we finish if we show that \(J\) is in fact a proper ideal.
    Suppose by way of contradiction that this fails. This means that there are finitely many functions \(\{g_i:i<n\}\) in \(F\) and a set \(X\in I\) such that
    \begin{equation*} A = X \cup \bigcup_{i<n}<(h, g_i). \end{equation*}
    Translating this into quantifiers, what we have is a function \(h\in\prod f\) such that
    \begin{equation} (\forall^I i\in A)(\exists i< n)\left[h(a)<g_i(a)\right] \tag{2.1.1} \end{equation}
    We have assumed that \(F\) is \(\leq_I\)-directed, so there is a single \(g\in F\) such that \(g_i\leq_I g\) for all \(i<n\text{.}\) But then (2.1.1) implies that \(h<_I g\) and this contradicts the choice of \(h\text{.}\) Thus \(J\) is a proper ideal on \(A\text{,}\) and \(h\) witnesses that \(F\) is \(\leq_J\)-bounded in \(\prod f\text{.}\) ◻

    Proposition 2.1.5.

    Let \(\ov{f}=\langle f_\alpha:\alpha<\lambda\rangle\) be a \(<_I\)-increasing sequence of functions.
    1. If \(\ov{f}\) has a least upper bound, then it is unique modulo \(I\text{.}\)
    2. If \(\ov{f}\) has a least upper bound modulo \(I\) that is not exact, then there is a function \(g\in\On^A\) for which the \(\subseteq_I\)-increasing sequence \(\langle t_\alpha:\alpha<\lambda\rangle\) of subsets of \(A\) defined by letting \(t_\alpha = \{a\in A: g(a)<f_\alpha(a)\}\) does not stabilize modulo \(I\text{.}\)

    Proof.

    Clause (1) is easy, as if \(f\) and \(g\) are upper bounds for \(\ov{f}\) then so is the function \(\min(f, g)\text{.}\) Suppose now that \(f\) is a least upper bound for \(\ov{f}\) that is not exact. This means that \(\ov{f}\) fails to be cofinal in \((\prod f, <_I)\text{,}\) and so there is a function \(g\in\prod f\) that witnesses this. We claim now that \(g\) has the required properties. It is obvious that sequence \(\langle t_\alpha:\alpha<\lambda\rangle\) will be \(\subseteq_I\)-increasing because \(\ov{f}\) is \(<_I\)-increasing. Moreover, for each \(\alpha<\lambda\) both \(t_\alpha\) and its complement must be \(I\)-positive: \(g\) does not bound \(\ov{f}\) modulo \(I\text{,}\) nor does \(\ov{f}\) bound \(g\text{.}\) If the sequence of sets \(t_\alpha\) were to eventually stabilize modulo \(I\text{,}\) say \(t_\alpha =_I B\) for all sufficiently large \(\alpha<\lambda\text{,}\) then we have a contradiction of part (2) of Theorem 2.1.4 because \(\ov{f}\) is bounded by \(g\) on \(A\setminus B\text{.}\) ◻
    The following corollary follows from (2) above by an easy argument which we relegate to an exercise.

    Corollary 2.1.6.

    If \(\ov{f}=\langle f_\alpha:\alpha<\lambda\rangle\) has a least upper bound that is not exact, then there is a family \(\langle X_\alpha:\alpha<\lambda\rangle\) of \(I\)-positive sets such that \(X_\alpha\cap X_\beta \in I\) for distinct \(\alpha\) and \(\beta\) less than \(\lambda\text{.}\)

    Subsection 2.1.2 Examples

    We finish this section with a few examples chosen to illustrate the definitions we have been investigating.

    Example 2.1.7.

    Let \(A=\omega\) and let \(I\) be the ideal \(\Fin(\omega)\) of finite subsets of \(\omega\text{.}\) For each \(n<\omega\) let \(f_n\) be constant with value \(n\text{,}\) and let \(F =\{f_n:n<\omega\}\text{.}\) If \(g\) is any \(\leq_I\)-upper bound for \(F\) in \(\On^A\text{,}\) the function \(h\) defined by
    \begin{equation*} h(n) = \begin{cases} g(n)-1 &\text{if $g(n)>0$, and}\\ 0 &\text{otherwise.} \end{cases} \end{equation*}
    is another upper bound for \(F\) with \(h<_I g\text{.}\) Even though \(F\) is \(\leq_I\)-bounded in \(\On^A\text{,}\) it does not have a minimal upper bound. \(\blacksquare\)

    Example 2.1.8.

    Continuing with \(A\) and \(I\) as in the previous example, let \(f_\alpha\) be constant with value \(\alpha\) for each \(\alpha\leq\omega_1\text{.}\) We claim that \(g=f_{\omega_1}\) is an exact upper bound for \(F=\{f_\alpha:\alpha<\omega_1\}\) modulo \(I\text{.}\) Suppose \(h<_I g\text{,}\) so \(h(n)<\omega_1\) for all sufficiently large \(n\text{.}\) Since \(|A|<\cf(\omega_1)\text{,}\) there is a single \(\alpha<\omega_1\) such that \(h(n)<\alpha\) for all sufficiently large \(n\text{,}\) hence \(h<_I f_\alpha\) as required. \(\blacksquare\)

    Example 2.1.9.

    Let \(A=\omega_1\text{,}\) and for each \(\alpha<\omega_1\) let \(f_\alpha\) be the function that is constant with value \(\alpha\) for \(\alpha<\omega_1\text{.}\) Let \(F=\{f_\alpha:\alpha<\omega_1\}\text{,}\) and let \(g\) be the identity function on \(\omega_1\text{.}\) What occurs modulo the ideal \(I=J^\bd[A]\) of bounded subsets of \(A\text{?}\) The function \(g\) is a minimal upper bound for \(F\) because any function \(h\in\prod g\) is bounded on a stationary (hence uncountable) set. On the other hand \(g\) is not a least upper bound bound for \(F\text{:}\) if \(X\) is the collection of countable successor ordinals, then the function \(h\) defined by
    \begin{equation*} h(\alpha) = \begin{cases} \beta &\text{if $\alpha=\beta+1$, and}\\ \alpha &\text{otherwise} \end{cases} \end{equation*}
    sits below \(g\) on an \(I\)-positive set, but is still an upper bound for \(F\) modulo \(I\text{.}\) \(\blacksquare\)

    Example 2.1.10.

    What occurs in the previous example if we enlarge \(I\) to be the non-stationary ideal on \(\omega_1\text{?}\) The function \(g\) is now a least upper bound for \(F\) because of the normality of \(I\text{:}\) any function that is less than \(g\) on an \(I\)-positive set is constant on an \(I\)-positive set and therefore cannot be an upper bound for \(F\) modulo \(I\text{.}\) On the other hand, \(g\) is not an exact upper bound for \(F\text{.}\) To see this, let \(\langle S_\alpha:\alpha<\omega_1\rangle\) partition \(\omega_1\) into disjoint stationary sets, and let \(T_\alpha = S_\alpha\setminus \alpha+1\text{.}\) Then each set \(S_\alpha\) is stationary, and we have arranged \(\alpha<\min(S_\alpha)\text{.}\) Now define a function \(h\) with domain \(A\) by setting
    \begin{equation*} h(\beta)= \begin{cases} \alpha &\text{if $\beta\in S_\alpha$, and}\\ 0 &\text{otherwise.} \end{cases} \end{equation*}
    The function \(h\) is in \(\prod g\) (as \(h(\beta)<\beta\) for all \(\beta\in A\)) but there is no \(\alpha<\omega_1\) such that \(h\leq_I f_\alpha\text{.}\) \(\blacksquare\)
