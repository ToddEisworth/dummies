    Section 2.2 Flat Products and Sequences

    This section begins with an expansion of Example [cross-reference to target(s) "ex:importanteub" missing or not unique], as the existence of an exact upper bound in that situation is a general phenomenon that crops up in many places.
    Suppose \(A\) is an infinite set and \(\lambda\) is a regular cardinal with \(|A|<\lambda\text{.}\) If \(\ov f=\langle f_\alpha:\alpha<\lambda\rangle\) is a \(<\)-increasing sequence in \(\On^A\text{,}\) then the pointwise supremum of the functions \(f_\alpha\) is an exact upper bound for \(\ov f\) modulo any proper ideal \(I\) on \(A\text{.}\) To see this, let \(g\) be the corresponding supremum, that is, \(g(a)=\sup\{f_\alpha(a):\alpha<\lambda\}\text{.}\) Let \(h\) be any function in \(\prod g\text{.}\) For each \(a\in A\text{,}\) we define \(\alpha_a\) to be least \(\alpha<\lambda\) for which \(h(a)\leq f_\alpha(a)\text{.}\) Clearly \(\alpha_a\) will exist, as the sequence \(\langle f_\alpha(a):\alpha<\lambda\rangle\) is strictly increasing with supremum \(g(a)\text{.}\) Since \(|A|<\lambda\text{,}\) we know the supremum of the ordinals \(\alpha_a\) (for \(a\in A\)) is some ordinal \(\alpha\) strictly less than \(\lambda\text{,}\) and therefore \(h(a)\) will be less than \(f_\alpha(a)\) for all \(a\in A\text{.}\) Since \(h\) is bounded everywhere by \(f_\alpha\text{,}\) it is clear that \(h<_I f_\alpha\) for any proper ideal \(I\) on \(A\text{,}\) and therefore \(g\) is an exact upper bound for \(\ov f\) mod \(I\text{.}\) \(\blacksquare\)
    The point is that the pointwise supremum of a sufficiently long (that is, of length strictly greater than the cardinality of \(A\)) \(<\)-increasing sequence will be an exact upper bound of the sequence for reasons of elementary cardinal arithmetic. Turning this around, it is easy to characterize when a function \(g\in\On^A\) will be an exact upper bound for such a \(<\)-increasing sequence.
    Suppose \(A\) is an infinite set and \(\lambda\) is a regular cardinal greater than \(|A|\text{.}\) If \(f\in\On^A\) and \(\cf(f(a))=\lambda\) for all \(a\in A\text{,}\) then \((\prod f, <)\) will have true cofinality \(\lambda\text{.}\)
    Proof. For each \(a\in A\) let \(c_a:\lambda\rightarrow g(a)\) be the increasing enumeration of a cofinal subset of \(f(a)\text{.}\) Then for each \(\alpha<\lambda\text{,}\) we can define
    \begin{equation*} f_\alpha(a):= c_a(\alpha), \end{equation*}
    and the sequence \(\ov{f}=\langle f_\alpha:\alpha<\lambda\rangle\) is \(<\)-increasing. Since \(f\) is the pointwise supremum of this sequence, we are in the situation of Example [cross-reference to target(s) "ex:longexact" missing or not unique]. Thus, \(f\) is an exact upper bound for \(\ov{f}\) and \((\prod f, <)\) has true cofinality \(\lambda\text{.}\) ◻
    Of course, it is sufficient to demand only that the cofinality of \(f(a)\) should be \(\lambda\) for \(I\)-almost all \(a\in A\text{,}\) as such a function is equal modulo \(I\) to one that has the required cofinality everywhere. The following proposition wraps Proposition [cross-reference to target(s) "prop:flatfunction" missing or not unique] and Example [cross-reference to target(s) "ex:longexact" missing or not unique] together, along with a couple of other easy equivalences.
    Suppose \(\lambda\) is a regular cardinal with \(|A|<\lambda\text{.}\) Then the following are equivalent for a function \(f:A\rightarrow\On\text{.}\)
    1. \(\cf(f(a))=\lambda\) for all \(a\in A\text{.}\)
    2. \(\prod f\) contains a \(<\)-increasing sequence \(F=\{ f_\alpha:\alpha<\lambda\}\) and \(f=\sup(F)\text{.}\)
    3. \(\tcf(\prod f, <)=\lambda\text{.}\)
    4. \(\tcf(\prod f, \leq_I)=\lambda\) for every (proper) ideal \(I\) on \(A\text{.}\)
    Proof. The implication from (1) to (2) holds using , and does not require \(\lambda\) to be greater than \(|A|\text{.}\) Moving from (2) to (3) is done as in Example [cross-reference to target(s) "ex:longexact" missing or not unique], and (4) follows from (3) by definition. If (1) fails, then it fails for some specific \(a\in A\text{,}\) and if we let \(I\) be the principle ideal \(\{a\}\) then \(\tcf(\prod f, <_I)\) will be \(\cf(f(a))\neq\theta\text{.}\) ◻
    If start with an ideal \(J\) and weaken condition (1) to require only that \(f(a)\) has cofinality \(\lambda\) for \(J\)-almost all \(a\in A\text{,}\) then we lose (2) and (3) in the above equivalence, but (4) remains equivalent if we require that the condition holds for all proper ideals \(I\) on \(A\) that extend \(J\text{.}\) Of course, every such \(f\) is equal (modulo \(J\)) to a function that attains cofinality \(\lambda\) in every coordinate. This configuration arises enough that it makes sense to assign some terminology.
    Let \(A\) be a set and let \(\lambda\) be a regular cardinal.
    1. A function \(f\in\On^A\) is said to be flat of cofinality \(\lambda\) if \(\cf(f(a))=\lambda\) for \(I\)-almost all \(a\in A\text{.}\)
    2. The product \(\prod f\) associated with such a function is called a flat product with cofinality \(\lambda\text{.}\)
    3. A set \(F\) of functions in \(\On^A\) is flat of cofinality \(\lambda\) modulo \(I\) if \(F\) has an exact upper bound \(f\) that is flat of cofinality \(\lambda\text{.}\)
    Our aim is to give an internal characterization of flat sets of functions, at least in the case where \(\lambda>|A|\text{.}\) We do this in Theorem [cross-reference to target(s) "flatEquivs" missing or not unique] below, but before stating and proving the theorem we will engage in a bit of exploration. Suppose that \(F\) is flat of cofinality \(\lambda\) modulo \(I\) with \(\lambda>|A|\text{,}\) and let \(f\) be a corresponding flat exact upper bound for \(F\text{.}\) We may assume that \(\cf(f(a))=\lambda\) for all \(a\in A\text{,}\) so we can also fix for each \(a\in A\) and unbounded \(S(a)\subseteq f(a)\) of order-type \(\lambda\text{.}\) The work we did in Example [cross-reference to target(s) "ex:longexact" missing or not unique] and Proposition [cross-reference to target(s) "prop:flatfunction" missing or not unique] shows us that \(F\) is equivalent modulo \(I\) to the product \(\prod_{a\in A}S(a)\text{,}\) and also equivalent modulo \(I\) to a \(<\)-increasing sequence \(\langle f_\alpha:\alpha<\lambda\rangle\text{.}\) This turns out to be a powerful assumption, as it implies the pointwise supremum of fewer than \(\lambda\) functions in \(F\) is bounded modulo \(I\) in \(F\text{:}\) Given \(F_0\subseteq F\) with \(|F_0|<\lambda\text{,}\) we choose for each \(g\in F_0\) a corresponding ordinal \(\alpha(g)<\lambda\) with \(g<f_{\alpha(g)}\text{.}\) Since \(\lambda\) is regular, we know that the supremum over all \(g\in F_0\) of the ordinals \(\alpha(g)\) will be less than \(\lambda\text{,}\) and hence \(\sup(F_0)\) is everywhere bounded by \(f_\alpha\) for some \(\alpha<\lambda\text{.}\) Now we can choose \(g\in F\) for which \(g_\alpha\leq_I g\text{.}\) Looking back to notation established in the last chapter, we have shown that \(\sigma_I(F)\) is at least \(\lambda\text{.}\) This observation is one of the important ingredients in our internal characterization of flatness.
    Suppose \(|A|<\lambda\) with \(\lambda\) regular. Then the following statements are equivalent for a family of functions \(F\subseteq\On^A\) and ideal \(I\) on \(A\text{:}\)
    1. \(F\) is flat of cofinality \(\lambda\) modulo \(I\text{.}\)
    2. \(\sigma_I(F)=\mathfrak{b}_I(F)=\mathfrak{d}_I(F)=\lambda\text{.}\)
    3. \((F, <_I)\) has a cofinal subset \(\{f_\alpha:\alpha<\lambda\}\) of cardinality \(\lambda\) such that
      \begin{equation*} \sup\{f_\beta:\beta<\alpha\}<_I f_\alpha \end{equation*}
      for all \(\alpha<\lambda\text{.}\)
    4. \(F\) is \(<_I\)-directed and equivalent modulo \(I\) to a \(\leq\)-increasing sequence of length \(\lambda\text{.}\)
    5. \(F\) is equivalent modulo \(I\) to a \(<\)-increasing sequence of length \(\lambda\text{.}\)
    6. \(F\) is equivalent modulo \(I\) to a product \(\prod_{a\in A}S(\alpha)\) where \(S(a)\) has order-type \(\lambda\text{.}\)
    Proof. If \(F\) is flat of cofinality \(\lambda\text{,}\) then by our discussion above we know \(\sigma_I(F)\) is at least \(\lambda\text{.}\) Since \(F\) has true cofinality \(\lambda\text{,}\) we can invoke Theorem I.[cross-reference to target(s) "thm:tcfchar" missing or not unique] and conclude
    \begin{equation*} \lambda\leq\sigma_I(F)\leq\mathfrak{b}_I(F)=\mathfrak{d}_I(F)=\lambda. \end{equation*}
    If we assume (2), then (3) follows from the proof of Theorem I.[cross-reference to target(s) "thm:tcfchar" missing or not unique]. Since \(\mathfrak{d}_I(F)=\lambda\) we fix a cofinal family \(\{g_\alpha:\alpha<\lambda\}\) in \(F\) and build \(\langle f_\alpha:\alpha<\lambda\rangle\) by induction, making sure at stage \(\alpha\) to select \(f_\alpha\) so that
    \begin{equation*} \max\{\sup\{f_\beta:\beta<\alpha\}, g_\alpha\}<_I f_\alpha. \end{equation*}
    Clearly (3) implies that \(F\) is \(<_I\)-directed. If we define
    \begin{equation*} h_\alpha:=\sup\{f_\beta:\beta<\alpha\} \end{equation*}
    then \(\langle h_\alpha:\alpha<\lambda\rangle\) is \(\leq\)-increasing and equivalent to \(F\text{.}\)
    How would we establish (4) from the existence of such a sequence \(\ov h\text{?}\) Given \(a\in A\text{,}\) look at the sequence \(\langle h_\alpha(a):\alpha<\lambda\rangle\text{.}\) This is non-decreasing by our assumption on \(\ov{h}\) and nothing in our assumptions prevents the sequence from stabilizing. However, our assumptions DO imply that this cannot happen very often: the set of \(a\) for which the sequence of values \(h_\alpha(a)\) eventually stabilizes must be in \(I\text{.}\)
    To see this, suppose it fails and the set \(X\) of \(a\in A\) for which the corresponding sequence stabilizes is \(I\)-positive. Since \(|A|<\lambda\) and \(\lambda\) is regular, there will be a \(\delta<\lambda\) with the property all of these sequences will have stabilized before stage \(\delta\text{.}\) Thus, if \(a\in X\) and \(\delta\leq\epsilon<\lambda\) we know \(h_\delta(a)=h_\epsilon(a)\) and hence \(f_\epsilon(a)\leq h(\delta(a))\text{.}\) But now we have a problem: since \(F\) is \(<_I\)-directed and equivalent to \(\ov{h}\) so we can find an \(\alpha\) greater than \(\delta\) such that \(h_\delta(a)<f_\alpha(a)\) for \(I\)-almost all \(a\) in \(A\text{.}\) Thus, the set \(X\) must be in \(I\text{.}\)
    Given \(\alpha<\lambda\text{,}\) if \(a\) is not in \(X\) then there will be some ordinal \(\beta(a)\) greater than \(\alpha\) for which \(h_\alpha(a)<h_{\beta(a)}(a)\text{.}\) We can use the fact that \(|A|<\lambda\) once more to conclude
    \begin{equation*} (\forall \alpha<\lambda)(\exists\beta<\lambda)(\forall a\in A\setminus X)\left[h_\alpha(a)<h_\beta(a)\right] \end{equation*}
    and now (5) follows easily. The implications from (5) to (6) and from (6) to (1) have been established earlier in this section. ◻
    We end this section by pointing out how the preceding theorem clarifies the relationship between simply having true cofinality \(\lambda\) modulo \(I\) and being flat of cofinality \(\lambda\) modulo \(I\text{.}\) Given a directed structure \((F, <_I)\) we know
    \begin{equation*} \sigma_I(F)\leq\mathfrak{b}_I(F)\leq\mathfrak{d}_I(F) \end{equation*}
    in all cases, while the structure has true cofinality \(\lambda\) if and only if
    \begin{equation*} \mathfrak{b}_I(F)=\mathfrak{d}_I(F)=\lambda. \end{equation*}
    The structure \((F, <_I)\) will be flat of cofinality \(\lambda\) if and only if we have the stronger condition
    \begin{equation*} \label{eq:flatness} \sigma_I(F)=\mathfrak{d}_I(F)=\lambda. \end{equation*}
    Thus, flatness of a set of functions can be characterized solely in terms of cardinal characteristics
