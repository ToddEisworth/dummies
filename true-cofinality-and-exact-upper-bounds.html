    Section 1.4 True cofinality and exact upper bounds

    Subsection 1.4.1 True cofinality

    We now return to the issue discussed at the beginning of the previous section and ask the natural question of whether we can identify when a subset \(F\) of \(\On^A\) is equivalent to a set of functions linearly-ordered by \(<_I\text{.}\) This is crucial for us, and the following definition is fundamental for pcf theory.

    Definition 1.4.1.

    Let \(I\) be an ideal on the set \(A\text{.}\) A set of functions \(F\subseteq\On^A\) is said to have true cofinality modulo \(I\) if \(F\) is equivalent modulo \(I\) to a set \(G\subseteq\On^A\) that is linearly-ordered by \(<_I\text{.}\)
    We will take some time to probe this definition. First, suppose that \(F\) has true cofinality modulo \(I\text{,}\) witnessed by the subset \(G\) of \(\On^A\text{.}\) Since we can without loss of generality replace \(G\) by a cofinal subset, we may assume that the order type of \(G\) is either \(1\) or an infinite regular cardinal.
    The case where \(G\) is a singleton \(\{g\}\) is not very interesting, as this happens if and only if \((F, \leq_I)\) has a maximum. Why? If \(F\equiv_I\{g\}\text{,}\) then there is an \(f\) in \(F\) such that \(g\leq_I f\text{,}\) and it follows easily that \(F\) has \(f\) as a maximum. On the other hand, if \(F\) has a maximum function \(g\) modulo \(I\text{,}\) then \(F\equiv_I\{g\}\text{,}\) and so \(F\) will have true cofinality in this degenerate sense.
    The interesting situation occurs when \(F\) is \(\equiv_I\)-equivalent to some \(<_I\)-increasing subset of \(\On^A\) whose length \(\lambda\) is an (infinite) regular cardinal. The following theorem characterizes when this occurs.

    Theorem 1.4.2.

    Suppose \(F\) is \(<_I\)-directed and \(\lambda\) is an infinite regular cardinal. Then the following statements are equivalent.
    1. \(F\) has true cofinality \(\lambda\) modulo \(I\text{.}\)
    2. \(\mathfrak{b}_I(F)=\mathfrak{d}_I(F)=\lambda\text{.}\)
    3. \(F\) contains a \(<_I\)-increasing and cofinal subset of order-type \(\lambda\text{.}\)

    Proof.

    Assume (1), and let \(\langle g_\alpha:\alpha<\lambda\rangle\) be a \(<_I\)-increasing sequence in \(\On^A\) equivalent to \(F\text{.}\) We first see that \(\lambda\leq\mathfrak{b}_I(F)\text{.}\) To see this, we need to see that any subset of \(F\) of cardinality less than \(\lambda\) is bounded, so assume \(\kappa<\lambda\) and \(f_\alpha\in F\) for each \(\alpha<\kappa\text{.}\) For each \(\alpha<\kappa\text{,}\) we can find \(\beta(\alpha)<\lambda\) such that
    \begin{equation*} f_\alpha\leq_I g_{h(\alpha)}. \end{equation*}
    Since \(\lambda\) is regular and \(\kappa<\lambda\text{,}\) we know
    \begin{equation*} \delta:=\sup\{\beta(\alpha):\alpha<\kappa\}<\lambda. \end{equation*}
    The assumption that \(F\equiv_I G\) provides us with a function \(f\in F\) with \(g_\delta\leq_I f\text{,}\) and therefore for any \(\alpha<\kappa\) we have
    \begin{equation*} f_\alpha\leq_I g_{h(\alpha)}<_I g_\delta\leq_I f. \end{equation*}
    Thus, any subset of \(F\) of cardinality less than \(\lambda\) is bounded and
    \begin{equation*} \lambda\leq \mathfrak{b}_I(F). \end{equation*}
    Next, we prove \(\mathfrak{d}_I(F)\leq\lambda\text{.}\) For each \(\alpha<\lambda\) we can find a function \(f_\alpha\in F\) such that \(g_\alpha\leq_I f_\alpha\text{,}\) and we claim that the set \(\{f_\alpha:\alpha<\lambda\}\) is cofinal in \((F,\leq_I)\text{.}\) Given \(f\in F\text{,}\) we can choose \(\alpha<\lambda\) with \(f\leq_I g_\alpha\text{,}\) and hence \(f\leq_I f_\alpha\) as well. Thus
    \begin{equation*} \lambda \leq\mathfrak{b}_I(F)\leq\mathfrak{d}_I(F)\leq\lambda \end{equation*}
    and (2) is established.
    Condition (3) follows from (2) by an easy inductive construction. To see that (3) implies (1), note that \(F\) is equivalent to any of its cofinal subsets. ◻
    As a corollary of the above, note that if \(F\) has true cofinality \(\lambda\) modulo \(I\text{,}\) then this \(\lambda\) is necessarily unique. This brings us to the following definition, a definition central to our work.

    Definition 1.4.3.

    Suppose \(F\) is \(<_I\)-directed. We say \(\lambda\) is the true cofinality of \(F\) modulo \(I\text{,}\) written \(\lambda=\tcf_I(F)\text{,}\) if
    \begin{equation*} \mathfrak{b}_I(F)=\mathfrak{d}_I(F)=\lambda. \end{equation*}
    Notice that we are justified in using the modifier “the” because if \(F\) has a true cofinality modulo \(I\text{,}\) then this true cofinality is uniquely determined as being the common value of \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\text{.}\) Moreover, if \(F\) and \(G\) are equivalent families of functions modulo \(I\text{,}\) then one of them has a true cofinality if and only if the other does, and the values of these true cofinalities are the same.

    Subsection 1.4.2 Exact Upper Bounds

    Having now characterized when a set of functions \(F\subseteq\On^A\) is equivalent (modulo an ideal \(I\)) to a \(<_I\)-increasing sequence of functions, we take a look at when \(F\) is equivalent to a product. We have already used the notion of product in our brief treatment of proper classes in the previous section, but we pause to give the official definition.

    Definition 1.4.4.

    Suppose \(g\) is a function in \(\On^A\text{.}\) We define the product of \(g\text{,}\) \(\prod g\text{,}\) by
    \begin{equation*} \prod g:= \prod_{a\in A} g(a) = \{f\in\On^A: (\forall a\in A)\left[f(a)<g(a)\right]\}. \end{equation*}
    Clearly \(\prod g\) is non-empty if and only if \(g(a)\) is non-zero for all \(a\in A\text{,}\) so we will implicitly assume this is the case whenever we use the notation \(\prod g\). Given such a function \(g\text{,}\) we address several natural questions about the structure of \(\prod g\text{.}\) First, note that it is easy to characterize when a product is directed under \(<_I\text{:}\)

    Proposition 1.4.5.

    If \(I\) is an ideal on the set \(A\text{,}\) then \(\prod g\) is \(<_I\)-directed if and only if \(g(a)\) is a (non-zero) limit ordinal for almost all \(a\) in \(A\text{.}\)

    Proof.

    Proof. Suppose \(g(a)\) is a limit ordinal for almost all \(a\in A\text{,}\) and let \(F\) be a finite subset of \(\prod g\text{.}\) If we define
    \begin{equation*} h(a) = \max\{f(a):f\in F\} + 1 \end{equation*}
    then \(h\) is equal modulo \(I\) to a function in \(\prod g\text{,}\) and clearly \(f<_I h\) for all \(f\in F\text{.}\)
    For the other direction, let \(X = \{a\in A: g(a)\text{ is a successor}\}\) and define a function \(f\in\prod g\) by
    \begin{equation*} f(a)= \begin{cases} g(a)-1 &\text{if $a\in X$, and}\\ 0 &\text{otherwise.} \end{cases} \end{equation*}
    If \(X\notin I\text{,}\) then there is no function \(h\in\prod g\) such that \(f<_I h\text{,}\) and so \(\prod g\) fails to be \(<_I\)-directed. ◻
    The question about when a product has true cofinality modulo an ideal turns out to have depths that we will plumb shortly. For now, we provide the following shallow answer that follows immediately from our earlier work.

    Proposition 1.4.6.

    Suppose \(I\) is an ideal on the set \(A\text{,}\) and \(g\in\On^A\) is such that \(\prod g\) is \(<_I\)-directed. Then the following are equivalent:
    1. \(\tcf_I(\prod g)\) exists and is equal to \(\lambda\text{.}\)
    2. \(\lambda\) is a regular cardinal and there is a \(<_I\)-increasing and cofinal sequence of length \(\lambda\) in \(\prod g\text{.}\)
    So in the above situation, \(g\) is an upper bound for the sequence \(\langle f_\alpha:\alpha<\lambda\rangle\) (as each \(f_\alpha\) is in \(\prod g\)), but \(g\) also has the property that any \(h<_I g\) must lie below some \(f_\alpha\) modulo \(I\text{.}\) This latter property says that \(g\) is a “smallest” upper bound for \(F\) in a strong sense, and this turns out to be a fundamental idea that we capture in the following definition.

    Definition 1.4.7.

    Let \(I\) be an ideal on \(A\text{,}\) and suppose \(F\subseteq\prod g\) for some function \(g\in\On^A\text{.}\) We say \(g\) is an exact upper bound of \(F\) modulo \(I\) if for all \(h:A\to \On\text{,}\) if \(h<_I g\) then there exists some \(f\in F\) such that \(h\le_I f\text{.}\)
    Exact upper bounds arise in several places when one considers products. For an easy example, note that if \(g\in\On^A\) is never zero, then \(g\) is an exact upper bound for \(\prod g\) modulo the smallest ideal \(\{\emptyset\}\text{,}\) and hence modulo any ideal \(I\) on \(A\text{.}\) This follows immediately, as a function \(f\) is in \(\prod g\) if and only if \(f<_{\{\emptyset\}} g\text{.}\) Just as easy, we have the following characterization of exact upper bounds in terms of equivalence.

    Proposition 1.4.8.

    Let \(I\) be an ideal on the set \(A\text{.}\) The following statements are equivalent for a non-empty \(F\subseteq\prod g\text{:}\)
    1. \(g\) is an exact upper bound for \(F\) modulo \(I\text{.}\)
    2. \(F\) is cofinal in \(\prod g\text{.}\)
    3. \(F\equiv_I\prod g\text{.}\)

    Proof.
