Example 1.5.5.

Suppose \(f:A\rightarrow\On\) is such that \(f(a)\) is a limit ordinal for all \(a\in A\text{,}\) and for each \(a\in A\) let \(\lambda(a)=\cf(f(a))\text{.}\)
If we let \(\ov{\lambda}\) denote \(\langle \lambda(a):a\in A\rangle\) then \(\ov{\lambda}\) is an \(A\)-indexed sequence of regular cardinals.
For each \(a\in A\text{,}\) we can choose a set \(S(a)\) cofinal in \(f(a)\) of order-type \(\lambda(a)\text{,}\) and it is clear that the products \(\prod S(a)\) and \(\prod\ov{\lambda}\) are order-isomorphic through the obvious correspondence.
Since \(\prod S(a)\) is cofinal in \(\prod f\text{,}\) it follows that \(\prod f\) is Tukey equivalent to the product \(\prod \ov{\lambda}\text{.}\)
Moreover, this situation persists if we move from \(\leq\) to a \(\leq_I\) for some ideal \(I\) on \(A\text{:}\) it is still the case that \((\prod\ov{\lambda}, \leq_I)\) is order-isomorphic to a cofinal subset of \((\prod f, \leq_I)\) and hence these two structures are Tukey equivalent.
