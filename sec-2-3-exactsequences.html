    Section 2.3 Global Structure of Exact Sequences

    Now that we can claim to understand the structure of flat families of functions, we move now to the more general problem of characterizing when a \(<_I\)-increasing sequence of functions \(\ov{f}\) has an exact upper bound \(f\) that need not be flat.

    Definition 2.3.1.

    A \(<_I\)-increasing sequence \(\ov f=\langle f_\alpha\mid \alpha<\delta\rangle\) in \(\On^A\) is called exact of cofinality \(\lambda\) if the sequence \(\ov{f}\) has an exact upper bound and \(\cf(\delta)=\lambda\text{.}\)
    Notice that any \(<_I\)-increasing sequence of limit length \(\delta\) has a cofinal subsequence of length \(\cf(\delta)\text{,}\) and hence we will usually restrict our attention to increasing sequences with length a regular cardinal. If we need further generality, we will follow the practice established in the last section and say that a \(<_I\)-increasing sequence of length some limit ordinal \(\delta\) is exact if it has an exact upper bound, and then the original sequence will be equivalent modulo \(I\) to an exact sequence of length \(\cf(\delta)\text{.}\)
    As far as eubs go, there are some things we can say about their structure without knowing much about the sequence \(\ov{f}\) at all. For example, the following relies only on the fact that the length of the sequence is an infinite cardinal.

    Proposition 2.3.2.

    Suppose \(\ov f\) is an exact sequence of length \(\lambda\text{,}\) and let \(g\) be an exact upper bound. Then \(g(a)\) is an infinite limit ordinal for \(I\)-almost all \(a\in A\text{.}\)

    Proof.

    It is obvious that \(g\) cannot equal 0 on an \(I\)-positive set: just the fact that \(f_0<_I f_1\) is enough to guarantee that \(f_1(a)\) must be greater than \(0\) for almost all \(a\in A\text{,}\) and hence the same must hold for \(g\text{.}\)
    Next, suppose by way of contradiction that there is an \(I\)-positive set \(X\) of coordinates \(a\) for which \(g(a)\) is a successor ordinal. Define a function \(h\) with domain \(A\) by
    \begin{equation*} h(a)= \begin{cases} g(a)-1 &\text{if $a\in X$, and}\\ 0 &\text{otherwise}. \end{cases} \end{equation*}
    Since \(h<_I g\text{,}\) we can choose \(\alpha<\lambda\) with \(h<_I f_\alpha\text{.}\) If \(a\) is in \(X\) and \(h(a)<f_\alpha(a)\text{,}\) then it must be the case that \(g(a)\) is less than or equal to \(f_\alpha(a)\text{.}\) Thus, there is an \(I\)-positive set of \(a\) for which \(g(a)\leq f_\alpha(a)\text{.}\) But this is absurd as we know \(f_\alpha<_I g\text{.}\) ◻
    The preceding proposition allows us to invoke Example 1.5.5 when discussing exact sequences and assume without loss of generality that the exact upper bounds have the form \(\ov{\lambda}=\langle\lambda(a):a\in A\rangle\text{,}\) where each \(\lambda(a)\) is a regular cardinal. Moreover, if we know that \(\lambda(a)\) is greater than \(|A|\) for \(I\)-almost every coordinate \(a\text{,}\) then by the argument of Example 1.5.6 \(\ov{f}\) is equivalent modulo \(I\) to an isomorphic copy of the product \(\prod\{\lambda(a):a\in A\}\text{.}\)
    Suppose \(\ov{f}\) is a \(<_I\)-exact sequence of length \(\lambda\) with exact upper bound \(\ov{\lambda}\text{.}\)
    1. If the sequence \(\ov{\lambda}\) is constant with some value \(\theta\) on an \(I\)-positive set, then either \(\theta=\lambda\) or \(\theta\leq|A|\text{.}\)
    2. \(\lambda(a)\leq \lambda\) for \(I\)-almost all \(a\in A\text{.}\)
    Proof. It suffices to show that if \(\theta\) is a regular cardinal with \(|A|<\theta\) and \(\theta\neq\lambda\text{,}\) then \(\{a\in A: \lambda(a)=\theta\}\in I\text{.}\) To do this, assume \(\theta\) is a regular cardinal greater than \(|A|\) and there is an \(I\)-positive set \(X\) on which \(\lambda(a)=\cf(g(a))\) is constant with value \(\theta\text{.}\) If we let \(J\) be the ideal \(I\restr X\text{,}\) then \(\ov{f}\) is will be (modulo \(J\)) flat of cofinality \(\theta\text{.}\) Since \(J\) extends \(I\) and \(\ov{f}\) is exact of length \(\lambda\) modulo \(I\text{,}\) we conclude that \(\theta\) must equal \(\lambda\text{.}\)
    For the second part of the theorem, assume by way of contradiction that there is an \(I\)-positive set \(X\subseteq A\) such that \(\lambda<\lambda(a)\) for all \(a\in X\text{.}\) Since each \(\lambda(a)\) is a regular cardinal, we know that for \(a\) in \(X\text{,}\) the set \(\{f_\alpha(a):\alpha<\lambda\}\) is bounded below \(\lambda\text{.}\) Thus, we can define a function \(h\in\prod\lambda(a)\) by setting \(h\) to be \(0\) outside of \(X\text{,}\) while for all \(\alpha<\lambda\) and \(a\in X\text{,}\) \(f_\alpha(a)\) is less than \(h(a)\) We get a contradiction, as there must be a \(\beta<\lambda\) for which \(h<_I f_\beta\text{,}\) but \(f_\beta(a)<h(a)\) for an \(I\)-positive set of \(a\in A\text{.}\) ◻
    The preceding theorem foreshadows a connection between the structure of an exact sequence \(\ov{f}\) and the nature of its exact upper bound \(\ov{\lambda}\text{.}\) To see how, let us first focus on the situation where the regular cardinals in the upper bound are (almost) all greater than the cardinality of \(A\text{.}\) In this situation, we know that \(\lambda\) must be greater than \(|A|\) as well, and if our sequence \(\ov\lambda\) is constant with value \(\theta\) on an \(I\)-positive set, then the only possibility is for \(\theta\) to be \(\lambda\text{,}\) the length of \(\ov{f}\text{.}\) Let \(X\) be the set of \(a\in A\) for which \(\lambda(a)=\lambda\) and set \(Y = A\setminus X\text{.}\) It is certainly possible for either of these sets to be empty, but we know that since they partition \(A\) into finitely many pieces, at least one of them must be \(I\)-positive. If \(Y\) is in the ideal \(I\text{,}\) then our sequence \(\ov{f}\) is flat of cofinality \(\lambda\text{.}\) If \(X\) is in the ideal \(I\text{,}\) then for any \(I\)-positive set \(Z\) the sequence \(\ov{f}\) is not flat modulo \(I\restr Z\text{.}\) Even if both \(X\) and \(Y\) are in \(I^+\text{,}\) we still have shown that assuming that \(|A|<\lambda(a)\) for all \(a\text{,}\) our index set \(A\) will into two pieces: on one of which \(\ov{f}\) looks flat of cofinality \(\lambda\) modulo \(I\text{,}\) and on the other where \(\ov{f}\) is “nowhere flat” modulo \(I\text{.}\)
    What happens we don’t assume that \(\lambda(a)\) is always greater than \(|A|\text{?}\) First, note that since \(I\) need not be maximal, there is nothing preventing the situation where the sequence \(\ov{\lambda}\) is constant with different values on disjoint \(I\)-positive sets. So for example, we could have an \(I\)-positive set on which our sequence looks flat of cofinality \(\lambda\text{,}\) and there could be another \(I\)-positive set where \(\ov{\lambda}\) is constant with some value \(\theta\neq\lambda\text{.}\) What we do know is that if the latter situation occurs, then \(\theta\leq |A|\text{,}\) so there is an \(I\)-positive set on which \(\ov{f}\) lives in a “small product”. Here, in contrast to the flat situation, things can be very sensitive to the underlying set theory, even in the simplest situation where both \(A\) and \(\theta\) are \(\omega\text{.}\)
    Suppose \(\lambda>\aleph_0\) is an arbitrary uncountable regular cardinal. Let \(V\) be a model of \(\zfc\) in which \(2^{\aleph_0}=\lambda\) and \(\ma\) holds. Let \(F=\omega^\omega\text{,}\) the Baire space of all functions from \(\omega\) to \(\omega\text{,}\) and let \(I\) be the ideal of finite subsets of \(\omega\text{.}\) Martin’s Axiom implies that the cardinals \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\) are equal to the continuum, and so \((F, <_I)\) will have true cofinality \(\lambda\text{.}\) Any sequence \(\ov{f}\) witnessing this will have the function that is constant with value \(\omega\) as exact upper bound. Thus, the exact upper bound does not seem to be connected with the sequence \(\ov{f}\) in this situation.
    If we do not insist on our examples involving \(\Fin(\omega)\) and instead allow larger ideals, then we can have (at least consistently) even worse behavior as the following example shows.
    It is consistent that \(2^{\aleph_0}\) is arbitrarily large, and for any uncountable regular \(\kappa\leq 2^{\aleph_0}\) there is an ideal \(I(\kappa)\) on \(\omega\) such that \(\omega^\omega\) has true cofinality \(\kappa\) modulo \(I(\kappa)\text{.}\) Again, the exact upper bound is exactly the same across all relevant \(\kappa\) — \(\ov{\lambda}\) is always the function with domain \(\omega\) that is constant with value \(\omega\) — but the lengths of the exact sequences in \(F\) realize all possibilities by varying the ideals involved.
    As interesting as products of small sets are, they are not particularly useful for us because imposes few restrictions on how they behave, and we officially adopt the attitude that products of sets of size at most \(|A|\) are intractable. If our exact sequence \(\ov{f}\) has an exact upper bound \(g\) such that \(\cf(g(a))\leq|A|\) on an \(I\)-positive set, then we will not be able to say much about the structure \((\prod g, <_I)\) without adopting additional set-theoretic axioms because it hides a copy of a product of small sets within. Thus, the following definition is quite natural.
    Suppose \(\ov{\lambda}=\langle \lambda(a):a\in A\rangle\) is a sequence of regular cardinals. We define \(\liminf_I\ov{\lambda}\) to be the least cardinal \(\mu\) such that there is an \(I\)-positive set of \(a\in A\) with \(\lambda(a)\leq\mu\text{.}\)
    The preceding definition is made without assuming anything about the product \(\prod \ov{\lambda}\) having true cofinality modulo \(I\text{,}\) and so it makes sense for any sequence \(\ov{\lambda}\) of regular cardinals. It gives us some handy language as well: if \(\liminf_I(\ov{\lambda})\) is not greater than \(|A|\text{,}\) then we know there is an \(I\)-positive set of coordinates on which our product looks like a product of small sets, and we have declared such things to be taboo. Thus, we will want to focus our attention on products \(\prod\ov{\lambda}\) for which \(\liminf_I\ov{\lambda}\) is greater than \(A\text{.}\) Our intent is to look at this cardinal in the context of general products of regular cardinals \(\prod\ov{\lambda}\text{,}\) and then return to questions related to exact sequences a little later.
    We will start with some elementary observations, so let us assume \(\ov{\lambda}\) is an \(A\)-indexed sequence of regular cardinals. First, Definition [cross-reference to target(s) "def:liminfdef" missing or not unique] will always make sense because \(I\) is a proper ideal, and
    \begin{equation*} \min\{\lambda(a):a\in A\}\leq\liminf{}_I\ov\lambda \leq\sup\{\lambda(a):a\in A\}. \end{equation*}
    This shows us that the easiest way to guarantee that \(\liminf_I\ov{\lambda}\) is greater than \(|A|\) is to insist that each \(\lambda(a)\) is greater than \(|A|\text{,}\) and we will see this hypothesis come to the forefront as we move deeper into pcf theory.
    Second, it is elementary to prove that \(\liminf_I\ov{\lambda}\) must be in the closure of the set \(\{\lambda(a):a\in A\}\text{.}\) This is quite useful, because it implies that if each \(\lambda(a)\) is greater than \(|A|\) and \(\ov\lambda\) does not stabilize on an \(I\)-positive set, then \(\liminf_I\ov\lambda\) must be singular of cofinality at most \(|A|\) (see Exercise [cross-reference to target(s) "exer:singular" missing or not unique]).
    Finally, Definition [cross-reference to target(s) "def:liminfdef" missing or not unique] extracts a cardinal characteristic from the sequence \(\ov{\lambda}\text{,}\) without looking at the associated product \(\prod\ov{\lambda}\text{.}\) This is a shortcoming, as it would be useful to know that this particular cardinal could be recovered from the product itself as that would give us a means of looking at a collection of functions and determining if it is hiding an intractable product of small sets. Luckily, it happens that it is tightly connected to the cardinal \(\sigma_I(\prod\ov{\lambda})\text{,}\) which is defined from the product.
    Suppose \(\ov{\lambda}=\langle \lambda(a):a\in A\rangle\) is a sequence of regular cardinals. Then
    1. \(\liminf{}_I\ov{\lambda}\le \sigma_I(\prod\ov{\lambda})\text{,}\) and
    2. if \(|A|\leq\liminf_I\ov{\lambda}\) then \(\sigma_I(\prod\ov{\lambda})=\liminf\hphantom{}_I \ov\lambda\text{.}\)
    Proof. Let \(\mu=\liminf_I\ov{\lambda}\text{,}\) and suppose \(F\) is a subset of \(\prod\ov{\lambda}\) of some cardinality \(\theta<\mu\text{.}\) Since the product is closed downward under \(<_I\text{,}\) we need to show that the function \(h:=\sup(F)\) is in the product modulo \(I\text{.}\) Clearly \(h(a)\leq\lambda(a)\) for all \(a\in A\text{,}\) and \(h(a)<\lambda(a)\) whenever \(\theta<\lambda(a)\) because each \(\lambda(a)\) is regular. The choice of \(\theta\) means that this holds for almost all \(a\in A\text{,}\) so \(\sup(F)\) is equal (modulo \(I\)) to some function in our product.
    Now suppose \(|A|\leq\mu\text{.}\) We need to produce a subset \(F\) of our product of cardinality \(\mu\) whose supremum is not in \(\prod\ov{\lambda}\text{.}\) But this is easy: For each \(a\in A\) with \(\lambda(a)\leq\mu\) and \(\alpha<\lambda(a)\) we add a function to \(F\) that has value \(\alpha\) on coordinate \(a\) and is identically \(0\) elsewhere. The cardinality of \(F\) is at most \(\mu\times|A|=\mu\text{,}\) and \(\sup(F)(a)=\lambda(a)\) for all \(a\in A\) with \(\lambda(a)\leq\mu\text{.}\) But this is an \(I\)-positive set, so \(\sup(F)\) is not in our product. ◻
    As a corollary, we end up with the desired characterization of \(\liminf_I\ov{\lambda}\) using the functions in the product \(\prod\ov{\lambda}\text{,}\) at least for the situation in which we will operate most of the time.
    If \(\ov{\lambda}\) is a sequence of regular cardinals each greater than \(|A|\text{,}\) then
    \begin{equation*} \sigma_I\bigl(\prod\ov{\lambda}\bigr)=\liminf{}_I\ov{\lambda}. \end{equation*}
    We can take the above observation and use it to obtain a nice counterpart to our characterization of flatness [cross-reference to target(s) "eq:flatness" missing or not unique] as the relation \(\sigma_I(F)=\mathfrak{d}_I(F)\text{.}\)
    Suppose \(\ov{\lambda}\) is an \(A\)-indexed sequence of regular cardinals each greater than \(|A|\) and let \(F\) be the product \(\prod\ov{\lambda}\text{.}\) Then the following statements are equivalent:
    1. \(\liminf_I(\ov{\lambda})\) is regular.
    2. There is an \(a\in A\) for which \(\lambda(a)=\liminf_I(\ov\lambda)\text{.}\)
    3. \(\sigma_I(F)=\mathfrak{b}_I(F)\text{.}\)
    Proof. Let \(\mu\) denote \(\liminf_I(\ov\lambda)\text{.}\) We know \(\mu\) is in the closure of the set of regular cardinals \(\{\lambda(a):a\in A\}\text{,}\) and our assumption implies that \(|A|\) is less than \(\mu\text{.}\) If \(\mu\) is regular, it cannot be a limit point of the range of \(\ov{\lambda}\text{,}\) and therefore must be an element of this set and so (2) holds.
    Now assume (2). The inequality \(\sigma_I(F)\leq\mathfrak{b}_I(F)\) holds automatically and \(\mu=\sigma_I(F)\) by Corollary [cross-reference to target(s) "cor:liminfsigma" missing or not unique], so it suffices to produce an unbounded subset of \(F\) with cardinality \(\sigma_I(F)\text{.}\) For each \(\alpha<\mu\text{,}\) define a function \(f_\alpha\) by
    \begin{equation*} f_\alpha(a)= \begin{cases} \alpha &\text{if $\lambda(a)=\sigma_I(F)$, and}\\ 0 &\text{otherwise.} \end{cases} \end{equation*}
    The definition of \(\liminf_I(\ov\lambda)\) tells us that (2) implies \(\lambda(a)=\mu\) for an \(I\)-positive set of coordinates \(a\text{,}\) and so \(f_\alpha\) will have the value \(\alpha\) on an \(I\)-positive set \(X\text{.}\) Since \(\sigma_I(F)=\liminf_I\ov{\lambda}\text{,}\) (2) also implies that \(\sigma_I(F)\) is a regular cardinal greater than the cardinality of \(A\text{.}\) Thus, the sequence \(\langle f_\alpha:\alpha<\lambda\rangle\) is flat of cofinality \(\sigma_I(F)\) modulo the (proper) ideal \(I\restr X\text{,}\) and furthermore the pointwise supremum \(f\) of the sequence will be an exact upper bound. By construction, the supremum \(f\) is equal to \(\lambda(a)\) on \(X\text{,}\) so any function \(h\in F\) is below \(f\) modulo \(I\restr X\text{.}\) This means that there will be an \(\alpha<\mu\) such that \(h(a)<f_\alpha(a)\) for \(I\)-almost all \(a\in X\text{,}\) and we conclude that set of functions \(f_\alpha\) is unbounded in \(F\) modulo \(I\text{.}\)
    The final implication from (3) to (1) is easy given our prior work: we know \(\liminf_I\ov{\lambda}\) will equal \(\sigma_I(F)\text{,}\) and we finish by way of Proposition I.[cross-reference to target(s) "prop:firstb" missing or not unique], which tells us that \(\mathfrak{b}_I(F)\) is a regular cardinal. ◻
    We now come to an important corollary that tells us that singular cardinals are always hiding just around the corner in our work.
    Suppose \(\ov{\lambda}\) is an \(A\)-indexed sequence of regular cardinals greater than \(|A|\) and let \(F\) denote the product \(\prod\ov{\lambda}\text{.}\)
    1. If \(\sigma_I(F)=\mathfrak{b}_I(F)\text{,}\) then there is an \(I\)-positive set \(X\) on which \(F\) is flat of cofinality \(\sigma_I(F)\text{.}\)
    2. If \(\ov{\lambda}\) does not stabilize on an \(I\)-positive set, then
      \begin{equation*} \cf(\sigma_I(F))\leq|A|<\sigma_I(F)<\mathfrak{b}_I(F)\leq\mathfrak{d}_I(F)\leq |F|. \end{equation*}
    Proof. The first follows from the equivalence of (2) and (3) in our theorem, and the second from the equivalence of (1) and (2) combined with earlier observations. ◻
    The above corollary tells us where to look if we are interested in finding exact sequences that are “nowhere” flat but do not live in an intractable product of small sets: we are going to need to find products \(F\) of regular cardinals satisfying \(\mathfrak{b}_I(F)=\mathfrak{d}_I(F)\) (so there will be a true cofinality modulo \(I\)) for which the cardinal \(\sigma_I(F)\) is singular and strictly less than \(\mathfrak{b}_I(F)\) (which must happen if we are to avoid an \(I\)-positive set on which \(F\) looks flat modulo \(I\)). The following gives us our first example of such an exact sequence, but it has the defect of making some cardinal arithmetic assumptions, a defect that will be removed later in this chapter when we show that such sequences are quite common.
    Assume \(\gch\) holds, and let \(F\) be the product \(\prod\ov{\lambda}\text{,}\) where \(\lambda(n)=\aleph_{n+1}\text{.}\) If we let \(I\) be the ideal \(\Fin(\omega)\) of finite subsets of \(\omega\text{,}\) then \(F\) is \(<_I\)-directed, and so
    \begin{equation*} \sigma_I(F)\le\mathfrak b_I(F)\le \mathfrak d_I(F)\le |F|=\aleph_{\omega+1}. \end{equation*}
    We see that \(\liminf_I(\ov{\lambda})\) is \(\aleph_\omega\text{,}\) and therefore \(\sigma_I(F)=\aleph_\omega\) as well by Corollary [cross-reference to target(s) "cor:liminfsigma" missing or not unique]. Since \(\mathfrak{b}_I(F)\) is regular and at least of cardinality \(\sigma_I(F)\text{,}\) we conclude that both \(\mathfrak{b}_I(F)\) and \(\mathfrak{d}_I(F)\) are \(\aleph_{\omega+1}\) and therefore \(F\) has true cofinality \(\aleph_{\omega+1}\) modulo \(I\text{.}\)
